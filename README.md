# PDF Summarizer

An AI-powered PDF summarizer built with Gemini API.

## What it does
Upload any PDF and get a clean summary in seconds. Choose between Concise, Detailed, Bullet Points, or ELI5 mode.

## Live Demo
https://pdf-summariser.netlify.app

## Built with
- HTML / CSS / JavaScript
- Google Gemini API
- Deployed on Netlify

## How to use
1. Get a free API key from aistudio.google.com
2. Paste your key into the tool
3. Upload a PDF
4. Pick a summary mode and hit Summarize
